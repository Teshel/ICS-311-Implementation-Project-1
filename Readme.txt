Operation & Reference

The program is run at the command line with one parameter, the filename to be read.
example:
$ java Implement1JSM data.txt

After that, the user is prompted with a menu and several options to select from it. To select an option, type its name into the terminal.



Readme.txt

To compile, just run javac with no extra options

$ javac Implement1JSM.java
